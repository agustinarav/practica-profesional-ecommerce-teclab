# Practica profesional: Adidas ecommerce 
Este repositorio forma parte del trabajo final de la Práctica Profesional de la Tecnicatura en Aseguramiento de la Calidad de Software del Instituto Técnico Superior TECLAB, y tiene fines académicos. Su objetivo es analizar y realizar actividades de testing sobre el sitio web oficial de [Adidas](https://www.adidas.com.ar/).

El sitio web de Adidas es una plataforma de comercio electrónico y comunicación de marca, orientada a la venta de productos deportivos, la difusión de colecciones, campañas y contenidos informativos, y la interacción con usuarios a través de diferentes funcionalidades digitales.


## El alcance de este trabajo se centra en:

- El análisis funcional del sitio web
- La comprensión de su comportamiento desde el punto de vista del usuario y del analista de calidad
- La ejecución de pruebas manuales. No se busca desarrollar ni modificar el código de la aplicación, sino evaluarla como producto desde el rol de QA

No se busca desarrollar ni modificar el código del sitio, sino evaluarlo como producto digital desde el rol de QA.
La documentación generada en este repositorio tiene fines educativos y busca aplicar conceptos fundamentales de aseguramiento de la calidad de software, tales como diseño de casos de prueba, testing exploratorio, elaboración de checklists, registro de evidencias y reporte de defectos, utilizando prácticas habituales en proyectos de QA y herramientas de uso común.

## Descripción general del sistema
El sitio web de Adidas es una aplicación web orientada al comercio electrónico y a la experiencia de marca. Permite a los usuarios navegar por distintas categorías de productos, realizar búsquedas, aplicar filtros, visualizar detalles de productos, gestionar un carrito de compras y completar procesos de compra en línea.

Además, el sistema ofrece funcionalidades complementarias como la gestión de cuentas de usuario, acceso a contenidos editoriales (blogs, lanzamientos, campañas), suscripción a comunicaciones y adaptación del contenido según el dispositivo desde el cual se accede.

La plataforma está diseñada para ofrecer una experiencia consistente, intuitiva y accesible, reduciendo fricciones durante la navegación y el proceso de compra. A través de su interfaz web, el usuario puede interactuar con múltiples secciones del sitio sin necesidad de conocimientos técnicos.

El sitio web de Adidas se ejecuta en un entorno web y está pensado para ser utilizado por una amplia variedad de usuarios, tanto en dispositivos de escritorio como móviles, actuando como un canal principal de venta y comunicación digital de la marca.
