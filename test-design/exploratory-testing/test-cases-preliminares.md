
Identificación para Checklist de TC principales para Prioridad Alta – Severidad Crítica

---


#### Test Suite: Acceso al sitio y navegación básica

TC- La página de inicio se carga correctamente

TC- El usuario puede navegar por las categorías del sitio

TC- El usuario accede a la ficha de detalle de un producto

---

#### Test Suite: Funcionamiento de la barra de búsqueda

TC- La barra de búsqueda es visible y accesible desde todas las páginas

TC- El campo de búsqueda está vacío por defecto

TC- Se muestran resultados correctos al buscar por descripción de producto

TC- Se muestran resultados correctos al buscar por características del producto

TC- El usuario es redirigido a la página de detalle del producto o a una lista de resultados relevantes

TC- Se muestra un mensaje adecuado al buscar un producto inexistente

TC- Se muestra el mensaje correcto al ingresar símbolos especiales en el campo de búsqueda

TC- Se muestra un mensaje de error al ingresar una combinación inválida de caracteres especiales

TC- La funcionalidad de búsqueda permite utilizar palabras clave válidas y filtros

TC- Comportamiento de la búsqueda cuando el campo se encuentra vacío

TC- Los resultados de búsqueda coinciden con la palabra ingresada

TC- Comportamiento del sistema cuando no se encuentran resultados

---

#### Test Suite: Ficha de producto

TC- El precio del producto se visualiza correctamente

TC- Las imágenes del producto se muestran correctamente

TC- La descripción del producto y sus variantes se muestran correctamente

TC- El stock del producto es visible o se muestra un mensaje de sin stock

TC- El producto puede ser agregado al carrito desde la ficha de producto

TC- El botón Agregar al carrito se encuentra disponible cuando hay stock

TC- Se muestra un mensaje adecuado al intentar agregar un producto sin stock

TC- La tabla de talles se encuentra disponible en la ficha de producto

TC- Se encuentra disponible el cálculo del costo de envío o la información de envío

TC- Se encuentra disponible el botón o la información sobre los medios de pago

TC- Se muestran recomendaciones de productos relacionados

TC- Se encuentra disponible la opción para unirse al club Adidas

---

#### Test Suite: Agregar al carrito

TC- El producto puede ser agregado al carrito desde la ficha de producto

TC- La cantidad seleccionada se refleja correctamente en el carrito

TC- La variante seleccionada del producto se refleja correctamente en el carrito

TC- El carrito se actualiza correctamente al agregar múltiples productos

TC- Se muestra el resumen correcto del producto agregado (nombre, precio, cantidad)

TC- Se puede acceder al carrito desde cualquier página después de agregar un producto

---

Test Suite: Carrito de compras

TC- Los productos agregados al carrito permanecen después de recargar la página

TC- Los productos agregados como invitado se mantienen al iniciar sesión

TC- Los productos permanecen en el carrito durante la sesión activa

TC- El carrito no excede la cantidad máxima de productos permitidos

TC- Los productos agregados se visualizan correctamente en el carrito

TC- La cantidad de productos puede modificarse desde el carrito

TC- Se pueden eliminar productos del carrito correctamente

TC- El resumen del carrito muestra nombre, precio y subtotal de cada producto

TC- El carrito se actualiza correctamente al modificar cantidades o eliminar productos

---

Test Suite: Inicio de checkout

TC- El usuario puede acceder al checkout desde el carrito

TC- El sistema valida que el carrito no esté vacío antes de iniciar el checkout

TC- El sistema valida que los productos tengan stock disponible antes de iniciar el checkout

TC- Se visualizan correctamente los datos de facturación, envío y métodos de pago

TC- Se aplica correctamente un cupón de descuento

TC- Se muestra la confirmación de la compra antes de finalizar el pedido

TC- El sistema pregunta si el usuario desea iniciar sesión antes de completar el checkout

---

Test Suite: Stock y disponibilidad

TC- Se muestra la disponibilidad del producto en la ficha

TC- El sistema indica correctamente cuando un producto no tiene stock

TC- El carrito no permite agregar más unidades que el límite disponible

TC- Se muestran mensajes claros sobre la disponibilidad del producto

---

Test Suite: Precios, impuestos y descuentos

TC- El precio del producto es correcto en la ficha y en el carrito

TC- Se aplican correctamente los descuentos disponibles

TC- Los impuestos se calculan y muestran correctamente
